### [01] Navigation

- LOW: isActive() uses startsWith() which could false-match overlapping route prefixes
- LOW: Repeated conditional class logic across 3 nav sections

### [02] Hero Section
**Files read:** HeroSection.jsx, PersistentHeroText.jsx, useSafariAutoplay.js
**Findings:**
- MEDIUM: Hero video path "/videos/Showreel 2021.mp4" hardcoded in component, not in siteConfig
- MEDIUM: useSafariAutoplay.js hook exists but is NOT used anywhere — dead code
- MEDIUM: Desktop hero height uses `calc(100svh - 5rem + 150px)` — 150px extra may cause overflow
- MEDIUM: Mobile nav links in hero are text-xs (12px) — below 44px tap target minimum
- LOW: Safari user agent detection is fragile (regex-based)
- LOW: Non-Safari unmute timer (2.5s) is arbitrary with no fallback

### [03] Film Cards
**Files read:** FilmCard.jsx, FeaturedFilmCard.jsx, CollapsedFilmCard.jsx
**Findings:**
- MEDIUM: FeaturedFilmCard silently swallows video play errors `.catch(() => {})`

- MEDIUM: FilmCard credits use array index as React key: `key={i}`

- LOW: CollapsedFilmCard layout thrashing — 3 forced reflows in quick succession

# Full Site Audit Report — 2026-03-30

## Executive Summary

Audited 28 files across the entire BABA portfolio codebase. The site has a solid foundation — well-structured components, good responsive patterns, and clean architecture. However, there are **2 critical issues**, **4 high-severity issues**, and **19 medium-severity issues** that need attention before production launch. The biggest gaps are: broken data (null video URL, missing thumbnails), no SEO/meta tags, missing modal focus trap, and a navigation active state that doesn't visually differentiate.

## CRITICAL (2)

### C1. Film #14 has null videoUrl — modal shows nothing
**File:** `src/data/films.js:355`
**Impact:** Clicking "Hommage Back to 1992 (Video No. 1)" opens modal with no video.
**Fix:** Provide correct Vimeo URL or hide the film.

### C2. Navigation active state invisible — opacity-100 on both active and inactive
**File:** `src/components/layout/Navigation.jsx:86,102,117`
**Impact:** Users cannot tell which page they're on. All nav items look identical.
**Fix:** Apply different opacity/weight/size to inactive items (e.g., `opacity-50` or `text-muted`).

---

## HIGH (4)

### H1. No per-page SEO — same title/meta on all pages
**Files:** `src/App.jsx`, `src/components/layout/Layout.jsx`
**Impact:** Browser tabs all show same title. Social shares are blank. Google can't index individual pages.
**Fix:** Add react-helmet-async, set per-page title/description/og:image from siteConfig.

### H2. No focus trap in FilmModal — Tab escapes behind overlay
**File:** `src/components/films/FilmModal.jsx`
**Impact:** WCAG 2.1 Level AA violation. Keyboard users can interact with content behind the modal.
**Fix:** Implement focus trap (trap-focus library or manual implementation).

### H3. siteConfig.seo object exists but is never used
**File:** `src/data/siteConfig.js:130-145`
**Impact:** SEO data was written but never wired to any component. Wasted effort.
**Fix:** Use it when implementing H1.

### H4. Source maps enabled in production build
**File:** `vite.config.js:40-41`
**Impact:** Exposes full source code to anyone opening DevTools. Security/privacy issue.
**Fix:** Set `sourcemap: false` (or `'hidden'` for error tracking only).

## MEDIUM (19)

### M1. 10 of 15 films have thumbnail: null
**File:** `src/data/films.js` (multiple lines)
**Impact:** FeaturedFilmCard applies `backgroundImage: url(null)` — invalid CSS. No visual fallback.
**Fix:** Add poster images for all films, or add a fallback/placeholder.

### M2. Hero video path hardcoded in HeroSection
**File:** `src/components/ui/HeroSection.jsx:216,253`
**Impact:** Path "/videos/Showreel 2021.mp4" is in component, not siteConfig. Hard to maintain.
**Fix:** Add `siteConfig.showreel.videoFile` and reference it.

### M3. useSafariAutoplay.js hook is dead code
**File:** `src/hooks/useSafariAutoplay.js`
**Impact:** Defined but never imported anywhere. HeroSection implements autoplay inline.
**Fix:** Delete the file or refactor HeroSection to use it.

### M4. FeaturedFilmCard silently swallows video play errors
**File:** `src/components/films/FeaturedFilmCard.jsx:31`
**Impact:** `.catch(() => {})` — no logging, no fallback UI. User sees broken video.
**Fix:** At minimum log the error; ideally show poster fallback.

### M5. All FeaturedFilmCard videos use preload="auto"
**File:** `src/components/films/FeaturedFilmCard.jsx:67`
**Impact:** All visible film videos start downloading immediately. Heavy on bandwidth.
**Fix:** Use `preload="metadata"` or IntersectionObserver for lazy loading.

### M6. FilmModal has no iframe error handler or timeout
**File:** `src/components/films/FilmModal.jsx:66-73`
**Impact:** If Vimeo fails to load, spinner spins forever.
**Fix:** Add onError handler and 15-30s timeout.

### M7. No image error handlers in PhotoGrid/ProjectCard
**Files:** `src/components/photos/PhotoGrid.jsx:75-89`, `ProjectCard.jsx`
**Impact:** Failed images show no feedback — just empty space.
**Fix:** Add onError handler with fallback state.

### M8. FloatingGalleryHero splits tagline without length check
**File:** `src/components/photos/FloatingGalleryHero.jsx:14`
**Impact:** If tagline has <3 words, renders `undefined`.
**Fix:** Add array bounds check or default values.

### M9. Lightbox body overflow save/restore conflicts with SmoothScrollContext
**File:** `src/components/ui/Lightbox.jsx:108-116`
**Impact:** Could break smooth scroll after closing lightbox if contexts disagree on overflow state.
**Fix:** Coordinate with SmoothScrollContext or use a shared overflow manager.

### M10. SmoothScrollContext setScrollY on every frame
**File:** `src/context/SmoothScrollContext.jsx:100`
**Impact:** 60 state updates/sec → unnecessary re-renders of all children.
**Fix:** Throttle to ~10fps or use ref-only pattern for scroll position.

### M11. Custom ease override not guaranteed to clean up on unmount
**File:** `src/context/SmoothScrollContext.jsx:261-264`
**Impact:** If scroll animation with custom ease is interrupted, subsequent scrolls use wrong ease.
**Fix:** Always restore ease in cleanup, not just on animation completion.

### M12. Desktop hero height includes extra 150px
**File:** `src/components/ui/HeroSection.jsx:209`
**Impact:** `calc(100svh - 5rem + 150px)` may overflow viewport, causing scroll jumps.
**Fix:** Verify design intent; remove if unnecessary.

### M13. Mobile hero nav links below 44px tap target
**File:** `src/components/ui/HeroSection.jsx:236-246`
**Impact:** text-xs (12px) links with gap-4 — too small for reliable mobile tapping.
**Fix:** Increase padding/size to meet 44px minimum.

### M14. FilmCard credits use array index as React key
**File:** `src/components/films/FilmCard.jsx:51-58`
**Impact:** Incorrect reconciliation if credits list changes.
**Fix:** Use `${credit.role}-${credit.name}` as key.

### M15. FilmCard credits gap-11 not responsive
**File:** `src/components/films/FilmCard.jsx:49`
**Impact:** 44px gap is large on mobile screens.
**Fix:** Use `gap-6 md:gap-11`.

### M16. tailwind.config.js font comments are wrong
**File:** `tailwind.config.js:30-36`
**Impact:** Comments say "Steps-Mono" and "American Typewriter" but fonts are Helvetica Now.
**Fix:** Update comments.

### M17. Font file paths inconsistent (camelCase vs spaces)
**File:** `src/styles/index.css:37,62`
**Impact:** "HelveticaNow" vs "Helvetica Now Text" — could cause 404s if files don't match.
**Fix:** Verify actual file names on disk match the CSS paths.

### M18. .nav-link CSS classes are dead code
**File:** `src/styles/index.css:343-362`
**Impact:** Never used in JSX — all nav styling is inline Tailwind.
**Fix:** Remove dead classes.

### M19. App.jsx comments mention /films/:slug route that doesn't exist
**File:** `src/App.jsx:14-20`
**Impact:** Documentation mismatch. No film detail pages exist (modal-only design).
**Fix:** Update comments to reflect current architecture.
